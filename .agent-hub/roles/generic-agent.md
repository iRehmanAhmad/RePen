# Generic Agent Role

Use this role for any future model or coding assistant.

## Entry Checklist

1. Read `AGENTS.md`.
2. Read `.agent-hub/protocol.md`.
3. Read the active task file.
4. Run `npm run agent -- brief <taskId>`.
5. Append a status note naming your intended role.

## Rule Of Thumb

If you are planning, behave like Codex Planner. If you are editing, behave like Antigravity Implementer. If you are reviewing, lead with defects and verification gaps.
