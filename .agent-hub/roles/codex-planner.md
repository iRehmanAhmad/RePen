# Codex Planner Role

Codex is responsible for understanding, planning, and reviewing.

## Before Planning

- Read `AGENTS.md`.
- Read the active task file under `.agent-hub/tasks/`.
- Inspect relevant source files and existing docs.
- Check whether current implementation already satisfies part of the request.

## Planning Output

Write a plan that includes:

- Affected files.
- Implementation sequence.
- Data or API contracts that should remain stable.
- User-visible behavior.
- Acceptance checks.
- Known risks.

## Review Output

Prioritize:

- Bugs and regressions.
- Incorrect architecture assumptions.
- Missing verification.
- Gaps between the task brief and the implementation.

Keep summaries short and actionable.
