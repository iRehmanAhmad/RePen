## Key Features

* **🎨 Advanced Brushes**:
  * **Pen**: Clean, standard digital ink.
  * **Calligraphy Pen**: Smooth, pressure-simulated brush ink.
  * **Highlighter**: Semi-transparent yellow highlighting with dynamic canvas opacity control.
  * **Laser Pointer**: Temporary, fading ink designed to point out screen details during presentations.
* **✍️ Text Tool**: Click and type directly on the screen with full keyboard input capture.
* **📐 Magic Shapes**: Auto-shapes tool with supporting shapes popover:
  * **Rectangle / Circle / Line**
  * **Freehand Arrow**: Instantly draw freehand directional arrows.
* **💡 Presenter Spotlight**: Dim the screen background and focus your audience's attention on your mouse pointer. Features real-time **Radius** and **Darkness** range adjustments.
* **🔍 Magnifier**: High-DPI aligned viewport magnifier to zoom in on any section of the screen in real-time.
* **📓 Notebook Boards**: Instantly swap the desktop view for custom whiteboard and chalkboard textures:
  * **Plain Whiteboard / Blackboard**
  * **Ruled Notebook Lines / Grid Graph / Music Staff**
  * Custom background color swatches.
* **💾 Sessions & PDF Export**: Save your whiteboard drawing pages and export them into high-quality PDFs.

## ⌨️ Keyboard Shortcuts Cheat Sheet

| Shortcut | Description |
| :--- | :--- |
| `Ctrl+Shift+P` | Toggle Desktop Pass-Through (Cursor/Move mode) |
| `Ctrl+Alt+H` | Toggle Toolbar Visibility (Hide/Show overlay) |
| `Ctrl+Alt+N` | Start a New Notebook Board Page |
| `Ctrl+Alt+S` | Save Notebook Drawing Session (`.rpen`) |
| `Ctrl+Alt+O` | Open/Load a Saved Session |
| `Ctrl+Alt+E` | Export Active Notebook to PDF |
| `Ctrl+Alt+Left` | Switch to the Previous Page |
| `Ctrl+Alt+Right` | Switch to the Next Page (or create one) |
