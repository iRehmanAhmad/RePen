# RePen Screenshots Guide

Place public-facing media assets for the GitHub README and release pages in this folder.

Recommended files:

- `hero-toolbar.png`
- `drawing-tools.gif`
- `board-modes.png`
- `spotlight-or-magnifier.png`

Recommended capture order:

1. Toolbar visible with a real desktop app open underneath.
2. Pen, highlighter, and laser pointer demo GIF.
3. Whiteboard, grid, ruled, or blackboard background view.
4. Spotlight or magnifier in a presentation scenario.

Recommended image guidelines:

- Use clear 16:9 captures when possible.
- Keep UI crisp and readable at GitHub README widths.
- Avoid cluttered desktop backgrounds.
- Show one strong use case per image.
