* **[Amanat - Note Taking App](https://github.com/iRehmanAhmad/Amanat)**: A dedicated markdown note-taking app designed for developers and freelancers to stay productive offline.

#### 🧠 Machine Learning & Computer Vision
* **[Sign Language Detection](https://github.com/iRehmanAhmad/Sign-Language-Detection-using-Yolov5)**: Real-time sign language interpreter built using customized YOLOv5 models.
